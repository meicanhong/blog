项目描述
iNet 旨在解决日渐繁琐的网络运维问题，通过对主流网络设备的配置解析和建模，形成统一的配置模型，提供完善的配置管理接口,并通过编排引擎实现对一系列复杂网络服务自动化场景的支持。
项目技术栈
SpringBoot 、网络、PostgreSQL、JPA、Kafka、Elasticsearch、gRPC、Zeebe、Groovy、TestFSM 
负责模块
● 负责 SDN 网络拓扑可视化和访问路径可视化
● 负责F5负载均衡设备纳管、数据展示和配置下发
● 负责主流网络设备的 NAT 解析展示和配置下发
● 负责 Api 类型网络设备的调研及开发与维护
技术亮点
● 基于图构建网络拓扑，根据节点之间的网络配置判断节点间连通性情况。基于最短路径分析算法，分析出拓扑内存在的网络 路径，并分析路径上节点的网络配置作出优化建议
● 实现多线程注解。利用反射机制，在方法上添加注解，即可将该方法自动交由线程池处理，并校验其结果
● 基于 JPA 实现动态搜索功能，可实现多字段间的精确或模糊搜索、对 JSON 体内某一具体字段值进行搜索、多字段的自定义排序的功能。
● 实现数据库动态搜索工具。封装 JPA 实现动态搜索功能，实现动态字段搜索、排序型，提供完善的
●
