## 使用建议

### 与 JD 对齐
阅读职位描述 (Job Description)，提取关键技术词，在简历中覆盖 80% 以上。

### 分级展示
- **精通：** 深度使用 2 年以上，可独立架构和解决复杂问题
- **熟练：** 使用 1 年以上，可独立完成开发任务
- **了解：** 有过实际项目经验，可在指导下使用

### 组合展示
不要孤立列举技术，按技术领域分组：

**前端技术栈：**
React 18 + TypeScript + Next.js + Tailwind CSS，熟练使用 Hooks、Context、Suspense 等特性

**后端技术栈：**
Node.js + Express + PostgreSQL + Redis，实现 RESTful API 和 GraphQL 接口

**工程化能力：**
Webpack/Vite 构建优化，GitHub Actions CI/CD，性能监控与优化

### 避免过时技术
除非岗位明确要求，否则不要列举过时技术：
- AngularJS (Angular 1.x)
- jQuery (除非维护老项目)
- Bower
- CoffeeScript
